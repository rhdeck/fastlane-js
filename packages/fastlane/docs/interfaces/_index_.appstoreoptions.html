{string} itcProvider The provider short name to be used with the iTMSTransporter to identify your team. This value will override the automatically detected provider short name. To get provider short name run pathToXcode.app/Contents/Applications/Application\ Loader.app/Contents/itms/bin/iTMSTransporter -m provider -u 'USERNAME' -p 'PASSWORD' -account_type itunes_connect -v off. The short names of providers should be listed in the second column

{any} runPrecheckBeforeSubmit Run precheck before submitting to app review

{any} precheckDefaultRuleLevel The default precheck rule level unless otherwise configured
